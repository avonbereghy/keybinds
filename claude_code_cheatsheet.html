Claude Code Cheat Sheet — CLI Shortcuts · Slash Commands · Launch Flags · Config
Controls & Input

General Controls

Ctrl+CCancel input or generation
Ctrl+DExit session (EOF)
Ctrl+LClear screen (keeps conversation)
Ctrl+OToggle verbose output
Ctrl+RReverse search command history
Ctrl+V / Cmd+VPaste image from clipboard
Ctrl+BBackground a task (2x in tmux)
Ctrl+FKill background agents (2x to confirm)
Ctrl+TToggle task list
Ctrl+GOpen prompt in external editor
↑ / ↓Navigate command history
← / →Cycle dialog tabs

Mode Switching

Esc EscRewind / undo last action
Shift+TabCycle permission modes
Opt+P / Alt+PSwitch model
Opt+T / Alt+TToggle extended thinking

Multiline Input

\ + EnterAll terminals
Opt+EntermacOS Terminal.app
Shift+EnteriTerm2, WezTerm, Ghostty, Kitty
Ctrl+JAll terminals

Text Editing (Readline)

Ctrl+A / EBeginning / end of line
Alt+B / FMove back / forward one word
Ctrl+KDelete to end of line
Ctrl+UDelete entire line
Ctrl+WDelete word backward
Ctrl+YPaste (yank) deleted text
Alt+YCycle kill ring (after Ctrl+Y)

Quick Prefixes

/Invoke slash command or skill
!Execute bash directly (e.g. !git status)
@File path autocomplete

macOS: Set Option as Meta key in Terminal → Settings → Profiles → Keyboard for Alt shortcuts.

CLI & Config

Launch Commands

claudeStart interactive session
claude "query"Start with initial prompt
claude -p "query"Non-interactive: print and exit
claude -cContinue most recent conversation
claude -r "session"Resume specific session
claude -m opusUse a specific model
claude --effort maxMaximum thinking effort
claude --fast onFast mode
claude -wStart in isolated git worktree
cat f | claude -p "q"Process piped content

Useful Flags

-n nameSet session display name
--agent nameUse specific subagent
--add-dir pathAdd working directories
--max-turns NLimit agentic turns
--max-budget-usd $Max spend before stopping
-vVerbose turn-by-turn output
--debug [filter]Debug mode (e.g. "api,mcp")
--permission-modedefault / plan / dontAsk / bypass
--system-prompt textReplace default system prompt
--remote taskCreate web session on claude.ai
--from-pr PRResume from GitHub PR
